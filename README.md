# Interest Calculator

A simple and elegant interest calculator built with HTML, CSS, and JavaScript. This project allows users to calculate both simple and compound interest with an intuitive user interface.

## Features

- **Simple Interest Calculation**: Calculate simple interest using the formula SI = (P * R * T) / 100
- **Compound Interest Calculation**: Calculate compound interest using the formula A = P(1 + R/100)^T
- **Real-time Validation**: Input validation to ensure positive values
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Modern UI**: Beautiful gradient background with smooth animations
- **Quick Reset**: One-click reset to clear all fields and start over

## How to Use

1. **Enter Principal Amount**: Input the initial amount of money (in dollars)
2. **Enter Rate of Interest**: Input the annual interest rate (in percentage)
3. **Enter Time Period**: Input the time duration (in years)
4. **Select Interest Type**: Choose between Simple or Compound interest
5. **Calculate**: Click the Calculate button to see the results
6. **View Results**: See the calculated interest amount and total amount
7. **Reset**: Click Reset to clear all fields and start over

## Technical Details

### HTML
The HTML file defines the structure with:
- Input fields for principal, rate, and time
- Select dropdown for interest type
- Calculate and Reset buttons
- Result display section

### CSS
The stylesheet includes:
- Modern gradient background
- Card-based layout design
- Responsive design for mobile devices
- Smooth transitions and animations
- Focus states for better UX

### JavaScript
The script provides:
- Form input validation
- Interest calculation logic (both simple and compound)
- DOM manipulation for displaying results
- Event listeners for user interactions
- Enter key support for quick calculation


3. Open `index.html` in your web browser
